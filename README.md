# RecordSheet
Double entry accounting application for your intranet.

## Synopsis

RecordSheet records financial transactions and generating reports simply and
easily.

## Featues

 - Double entry bookkeeping
 - Imports data from OFX files
 - Generates reports
 - Multiple users

## Installation

Run `setup.py`
